# Visual-image review

AI-assisted images are specifications only until a human reviews them. Confirm the image is original, accurate to the question, legible on mobile, free of embedded answer cues, and has descriptive alt text and a concise caption. Store approved local assets under `public/question-images/`, set the source record to `approved`, and reject any asset with a misleading scale, label, or visual implication. Trapwise does not call an image model or API at runtime.
